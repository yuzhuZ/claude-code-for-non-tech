# Skills

Skills are reusable prompt packages that give Claude Code specialized capabilities. They live in `~/.claude/skills/` and get triggered by slash commands or automatically when Claude detects a matching context.

## Skills I Use Daily

### grill-me

Stress-tests your plans and designs by interviewing you relentlessly — asking hard questions, probing assumptions, and exploring edge cases until you reach shared understanding.

**When I use it**: Before committing to a project direction, presenting a PRD, or making a significant product decision. It forces me to think through things I'd otherwise skip.

**Source**: [mattpocock/skills/grill-me](https://github.com/mattpocock/skills/tree/main/grill-me)

---

### project-creator

Sets up new projects with all the right structure automatically — folders, CLAUDE.md, README, index entries, and operations log entries in one shot.

**When I use it**: Every time I start a new project. Eliminates the boilerplate of setting up consistent project structure manually.

**Source**: [DaronVee/ccgg-project-creator-skill](https://github.com/DaronVee/ccgg-project-creator-skill)

---

### skills-factory

A meta-skill for creating your own production-ready skills. Uses evaluation-driven development and progressive disclosure patterns to produce well-structured, validated skills.

**When I use it**: When I want to build a new custom skill. It guides the process from idea to working implementation.

**Source**: [DaronVee/ccgg-skills-factory](https://github.com/DaronVee/ccgg-skills-factory)
