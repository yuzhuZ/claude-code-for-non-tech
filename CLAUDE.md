## PROJECT MISSION

Personal knowledge base for everything Yuzhu learns about Claude Code: CLI features, the skills system, hooks, the Agent SDK, the Claude API, and practical patterns discovered through daily use.

**Core Focus**:
- CLI: slash commands, permissions, keybindings, settings, IDE integration
- Skills: authoring, triggering, best practices
- Hooks: event types, use cases, configuration examples
- Agent SDK: building agents, tool use, MCP servers
- Claude API: models, prompt caching, tool use, batching, files
